# Vault chain front-end

[repo](https://github.com/Cryptorcheck/vaultChain)

### Start

##### Environment Check
``` shell
# node >= 22
node -v
```

##### Install Dependencies
``` shell
yarn install
```

##### Run Project
``` shell
yarn dev
```

##### Build Project
``` shell
yarn build
```
